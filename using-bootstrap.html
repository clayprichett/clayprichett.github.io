Lesson Number One: Use Containers.

Most Bootstrap elements should be in containers, which are just divs with the class container-fluid for full-width or container for not-full width.

You'll also need rows and columns, which are just divs with those names. The column numbers should add up to 12.

Lesson Number Two: Block Quotes

Block quotes have their own tag. It's written as blockquote. There should also be a footer for the author or speaker.

Lesson Number Three: Navbars

Navbars have their own tag. You guessed it, it's called navbar.
